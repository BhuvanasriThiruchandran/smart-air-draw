# Smart Air Draw 🎨

This is a hand gesture drawing app.

You can draw in the air using your hand and control drawings using gestures.

## Features
- Draw using hand ✋
- Change colors 🎨
- Clear screen 🧹
- Save drawing 💾

## Tech Used
- React
- MediaPipe
- JavaScript

## How to Run
npm install  
npm run dev

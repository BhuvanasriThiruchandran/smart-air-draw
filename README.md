# Smart Air Draw 🎨

A hand gesture based virtual drawing web app that allows users to draw in the air using real-time hand tracking.

## Features
- ✋ Hand gesture drawing
- 🎨 Custom colors
- 🧹 Clear canvas
- 💾 Save drawing
- 📱 Responsive UI

## Tech Stack
- React + Vite
- JavaScript
- MediaPipe (Hand Tracking)

## Description
Smart Air Draw is an interactive web application that uses computer vision to detect hand gestures and allows users to draw virtually in the air without touching the screen.
